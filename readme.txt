== Description ==

This plugin displays functional compounding investment calculator. It can be used how a hypothetical investment can grow over time. It allows partial compounding. 

== Installation ==

1. Upload investment-calculator.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the CSS classes in the admin screen in Plugins/Investment Calculator Configuration. This is optional, the calculator will work fine without any CSS classes.
4. Place `[compounding-calculator]` in the content of a page or post. The calculator will appear on the place of this tag

== Frequently Asked Questions ==
